# Advent of Code 2024 - Jour 2 (Java)
Ce projet est ma solution personnelle pour le défi **Advent of Code 2024 - Day 2**.  
Réalisé dans le cadre d’un exercice demandé pour un stage en développement fullstack.
#  Objectif
Analyser des rapports de niveaux afin de déterminer :
1. Combien de rapports sont "sûrs" (monotones avec des différences entre 1 et 3)
2. Combien peuvent le devenir en supprimant un seul niveau.
   
# Technologies utilisées
- Java 17
- Programmation orientée objet
- Lecture de fichiers avec `BufferedReader`
# Clone le repo
git clone https://github.com/Amir2899/AdventOfCode_Day2_Java.git
# Compile
javac src/Solution.java
# Exécute
java -cp src Solution < input.txt
